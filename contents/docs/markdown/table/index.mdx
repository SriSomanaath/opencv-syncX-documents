## API Endpoints & Functionality

| **Endpoint**     | **Method** | **Description** |
| --------------- | --------- | --------------------------- |
| `/api/process`  | `POST`    | Process image with selected AI model |
| `/api/models`   | `GET`     | Fetch available AI models |
| `/api/analytics` | `GET`    | Retrieve processing stats & metrics |
| `/api/upload`   | `POST`    | Upload images/videos for processing |
