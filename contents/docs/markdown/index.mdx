## Getting Started

To begin using OpenCV SyncX, follow these simple steps:

1. Clone the repository to your local machine:

```bash
git clone https://github.com/opencv/syncx.git
cd syncx
```

2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

3. Start the server:

```bash
python syncx_server.py
```

4. Open the application in your browser:

```bash
http://localhost:5000
```
