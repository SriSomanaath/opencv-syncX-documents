## Getting Started with OpenCV SyncX

To get started with OpenCV SyncX:

1. **Sign in** to the **SyncX Dashboard**.
2. **Upload images/videos** or provide an API URL for processing.
3. **Choose AI models** for **detection, enhancement, or segmentation**.
4. **View real-time analytics & download results**.
5. **Automate workflows** via the SyncX **REST API**.

> **Tip:** The SyncX dashboard supports **drag-and-drop uploads**, **batch processing**, and a powerful **model selector** for customization.
